// Stat Definitions — from The Sports Statistics Manual by Chris Cain
OVR MLB
Overall Rating
A 0–99 composite score calculated from a pitcher's RPI (Runs Per Inning) or a batter's weighted offensive production relative to league average across the current season.
RPI MLB
Runs Per Inning (Pitchers)
Earned runs allowed divided by innings pitched. The primary metric in Chris's pitching rating system — lower is better. Used instead of ERA to better account for partial-inning performance.
ERA MLB
Earned Run Average
(Earned Runs × 9) ÷ Innings Pitched. The traditional pitching benchmark; RPI refines this for fractional innings.
FIP MLB
Fielding Independent Pitching
An ERA-like metric using only outcomes the pitcher controls: strikeouts, walks, hit-by-pitches, and home runs. Removes fielding from the equation.
WHIP MLB
Walks + Hits per Inning Pitched
(BB + H) ÷ IP. Measures baserunners allowed per inning. Below 1.00 is elite; above 1.40 is poor.
xFIP MLB
Expected FIP
Like FIP but replaces actual home runs with expected home runs based on fly ball rate. Better for predicting future ERA.
WAR All
Wins Above Replacement
Combines batting, baserunning, fielding (for position players) or pitching (for pitchers) into a single value of wins contributed above a replacement-level player.
OPS MLB
On-Base Plus Slugging
On-base percentage + slugging percentage. The most predictive single batting stat for run production. League average is typically around .720.
